# Y&Y Chinese and Thai Food - Website

A professional, mobile-responsive website for Y&Y Chinese and Thai Food, a family-owned food truck in Lowell, Arkansas.

## Project Details

**Business:** Y&Y Chinese and Thai Food  
**Location:** 405 S Bloomington St, Lowell, AR 72745 (Food Truck Park)  
**Phone:** (479) 238-4156  
**Rating:** 4.6/5 (92+ Google reviews)

## Features

✅ **Responsive Design** - Works seamlessly on mobile, tablet, and desktop  
✅ **Hero Section** - Eye-catching introduction with clear call-to-action  
✅ **About Section** - Family story and values highlighting  
✅ **Customer Reviews** - Real Google reviews showcasing quality  
✅ **Menu Highlights** - Featured dishes with pricing  
✅ **Location & Hours** - Integrated Google Map and business hours  
✅ **Easy Contact** - One-click phone call and social media links

## Deployment

**Option 1: Netlify (Recommended)**
1. Create account at netlify.com
2. Drag & drop the `yy-chinese-thai` folder
3. Site goes live instantly with random URL
4. Can add custom domain later

**Option 2: Manual Upload**
1. FTP/SFTP upload to hosting
2. Point domain to hosting server
3. Access via yourdomain.com
